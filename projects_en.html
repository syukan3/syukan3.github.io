Censify

Censify is an API service that detects inappropriate expressions from user-input text and returns their presence and similarity scores. It enables high-accuracy and efficient filtering of inappropriate expressions in UGC (User Generated Content) such as SNS and chat services.

Service URL: https://censify-lp.pages.dev

Key Features:

  • High-accuracy inappropriate expression detection using LLM
    • Integration of inappropriate list and whitelist in system prompt
    • Flexible judgment based on similarity (threshold ≥0.5 for inappropriate)
    • False detection prevention using whitelist
  • Fast API delivery
    • Edge computing with Cloudflare Workers
    • Low latency ensured at edge locations worldwide
  • Scalability and flexibility
    • Multi-language support (Japanese, English, and other major languages)
    • Custom dictionary feature for user-specific requirements
  • Security measures
    • Enforced HTTPS communication and API key authentication
    • JWT authentication for management API
    • Access log audit functionality

Tech Stack:

  • Frontend: HonoJS (TypeScript)
  • Infrastructure: Cloudflare Workers
  • AI/ML: Large Language Model (LLM)

Development Period: 2 months (Ongoing updates)

OmniOCR

OmniOCR is an AI OCR service that extracts text from images and PDFs and converts it into structured data. Simply upload images of invoices or receipts, and they are instantly converted into structured data, easily integrated through API or Webhook.

Service URL: https://omni-ocr-lp.pages.dev/

Key Features:

  • High-accuracy OCR
    • High-precision character recognition using latest AI technology
    • Multi-language support (Japanese, English, and other major languages)
  • Flexible Data Structuring
    • Free mapping of extracted data
    • Conversion to structured data like JSON
  • API and Webhook Integration
    • Provision of RESTful API and Webhook
    • Easy integration with other systems
  • Secure Management
    • Encrypted data storage
    • Strict access control

Tech Stack:

  • Frontend: Next.js (TypeScript)
  • Infrastructure: Cloudflare Pages
  • AI/ML: Large Language Model (LLM)

Development Period: 2 months (Ongoing updates)
