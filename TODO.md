# 残タスク（宿題）

ポートフォリオ残タスクのうち、素材の用意が必要なもの。素材が揃い次第サイトに反映する。
（記録日: 2026-06-11）

## ① 生成AIガバナンス事例の定量数値

対象: Work「生成AIの全社導入とガバナンス整備」（`data/works.json` の `work-genai-governance`、英語版は `data/en/works.json`）

公開できる範囲の実数を用意する（概数・「約」「◯%以上」表記でOK。実在する数値のみ）。

- [ ] 仕様書自動生成: 1本あたり作成時間 ◯時間 → ◯分、対象システム数
- [ ] 通話録音のLLM評価: 月の対象件数、人手評価工数 ◯h → ◯h
- [ ] 問い合わせ一次回答: 回答までの時間 ◯ → ◯
- [ ] 全社導入の規模: 利用部門数・利用率・月間プロンプト数など
- [ ] コスト/工数の総括: 「年間◯時間削減」「◯人月相当」「外注費◯万円削減」のいずれか1つでも
- [ ] 監査: 内部監査・FISCアセスの指摘是正件数、「指摘ゼロ継続◯年」など

注意:
- 金融業の社内数値のため、公開前に社内確認の要否を判断する
- 生数値が出せない場合は「◯割削減」「数百時間規模」のような丸め表現で掲載する

## ② 推薦の声（testimonials）

掲載時は新セクション（Home または About）を実装する。

- [ ] 誰から: 役職・関係性（実名掲載 or 「金融系企業 役員」等の匿名表記）
- [ ] 本文: 2〜4文（日本語でOK。英語版への翻訳は実装時に対応）
- [ ] 掲載許可: 本人の了承を確認
- 形式は自由（本人からのテキスト、LinkedIn推薦文の転載許可など）
