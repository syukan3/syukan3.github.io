MosaicAI

MosaicAIは、複数の最先端AI言語モデル（ChatGPT、Claude、Gemini、Perplexity）を統一的なインターフェースで利用できるオープンソースのPythonライブラリです。

主な特徴：

  • 複数AI言語モデルを簡単に利用できる統一されたインターフェース
  • API Keyの安全な管理と再利用
  • 拡張性を持つ設計で、新しいAIモデルの追加が容易
  • テキスト生成、画像説明生成、JSON生成機能
  • 詳細なドキュメントとサンプルコードの提供

MosaicAIはオープンソースプロジェクトとして開発されており、コミュニティからの貢献を歓迎しています。バグ修正、新機能の提案、ドキュメントの改善など、さまざまな形で参加できます。

GitHub Repository: https://github.com/syukan3/MosaicAI

開発期間：2ヶ月（継続的に更新中）

画像フィルターアプリ

PythonとOpenCVを使用した画像処理アプリケーション。ユーザーがアップロードした画像に様々なフィルターや効果を適用できます。

主な機能：

  • 画像のアップロードとプレビュー
  • 複数のフィルター（白黒、セピア、ぼかしなど）
  • 画像の回転、反転、リサイズ
  • 処理済み画像のダウンロード
  • AWS S3を使用した画像の保存と管理

開発期間：2ヶ月

売上分析ダッシュボード

PythonとReactを使用したデータ可視化ツール。企業の売上データをリアルタイムで分析し、わかりやすいグラフで表示します。

  • 製品カテゴリ別の売上比率
  • 地域別の売上ヒートマップ
  • 顧客セグメント分析
  • カスタマイズ可能なダッシュボードレイアウト
