Gaugenix

Gaugenixは、IoTデバイスを用いて計器の画像を撮影し、AI解析によってその値を取得するシステムです。工場やプラントなどの現場において、人手に頼っていた計器の読み取り作業を自動化し、データ収集の効率化と人為的なミス削減を実現します。

例えば、温度計や圧力計などのアナログ計器の画像を、設置されたカメラで定期的に撮影します。撮影された画像は、AIモデルによって解析され、計器の目盛りを読み取り、数値データに変換されます。取得されたデータは、クラウド上に保存され、ユーザーインターフェースを通じて確認したり、他のシステムと連携したりすることができます。

主な機能：

  • リアルタイムでのデータ取得と分析
  • ユーザーインターフェースによる簡単な操作
  • データの可視化とレポート生成
  • クラウドストレージへのデータ保存
  • APIを通じた他システムとの連携

開発期間：1年3ヶ月（継続的に更新中）

Censify

Censifyは、ユーザーが入力したテキストから不適切な表現を検出し、その存在および類似度を返却するAPIサービスです。SNSやチャットサービスなどのUGC（ユーザー生成コンテンツ）における不適切表現のフィルタリングを、高精度かつ効率的に実現します。

サービスURL: https://censify-lp.pages.dev

主な特徴：

  • LLMを活用した高精度な不適切表現の検出
    • システムプロンプトに不適切リストとホワイトリストを組み込み
    • 類似度に基づく柔軟な判定（しきい値≥0.5で不適切と判定）
    • ホワイトリストによる誤検知防止機能
  • 高速なAPI提供
    • Cloudflare Workersによるエッジコンピューティング
    • 世界中のエッジロケーションで低レイテンシーを確保
  • 拡張性と柔軟性
    • 多言語対応（日本語、英語、その他主要言語）
    • カスタム辞書機能によるユーザー固有の要件対応
  • セキュリティ対策
    • HTTPS通信の強制とAPIキー認証
    • 管理APIにおけるJWT認証
    • アクセスログの監査機能

技術スタック：

  • フロントエンド：HonoJS（TypeScript）
  • インフラ：Cloudflare Workers
  • AI/ML：大規模言語モデル（LLM）

開発期間：2ヶ月（継続的に更新中）

岡山大学 大学院自然科学研究科

電気エネルギー・システム制御工学研究室

期間: 2013年4月 - 2015年3月

研究テーマ:

太陽光発電と燃料電池車を用いた住宅用分散型発電システムのエネルギー管理手法に関する研究

研究背景:
  • 地球温暖化対策として再生可能エネルギーの導入が進められている
  • 太陽光発電や燃料電池などの分散型電源は、CO2排出量の削減に貢献すると期待されている
  • 分散型電源は出力変動が大きく電力系統に悪影響を及ぼす可能性がある
  • 分散型電源を効果的に活用するためには、エネルギー管理システム(EMS)が必要
研究概要:
  • 燃料電池車と太陽光発電を用いた住宅用分散型発電システムの新しいエネルギー管理手法を提案
  • 提案手法の有効性を検証
  • 通常の日常運転だけでなく、緊急時運転にも適用可能
提案手法の特徴:
  • 通常運転時: 電力変動の抑制率と1日あたりの運転コストに基づいて評価
  • 緊急時運転: 運転可能日数によって評価
研究結果:
  • 提案手法は電力変動の40%を抑制
  • 通常の日常運転における運転コストを削減
  • 緊急時運転では、電力系統が復旧するまでシステムを運転可能
国際学会 論文発表:
  • 論文タイトル: "Energy management method for residential distributed generation system using photovoltaic cells and fuel cell vehicle"
  • 著者: Shunsuke Sakae; Akiko Takahashi; Jun Imai; Shigeyuki Funabiki
  • 発行元: IEEE
  • 発行年: 2014
  • 会議名: 2014 IEEE Industry Application Society Annual Meeting
  • 論文URL: https://ieeexplore.ieee.org/author/37085512035
